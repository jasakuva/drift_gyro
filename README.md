# Drift Gyro

## Overview
The Drift Gyro project aims to provide a comprehensive solution for tracking the drift of devices using gyroscopic sensors. This documentation outlines the project's purpose, features, installation instructions, usage examples, and contribution guidelines.

## Features
- Real-time drift tracking
- Calibration options for different sensors
- User-friendly interface

## Installation
To install the Drift Gyro project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/jasakuva/drift_gyro.git
   cd drift_gyro
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To start tracking drift, run the following command:
```bash
python main.py
```
You can configure settings in the `config.yaml` file.
